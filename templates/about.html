About Digit AI

  • Digit AI is a powerful and responsive handwritten digit recognition system built using deep learning.
  • The model is trained on the MNIST dataset using convolutional neural networks (CNNs) and achieves over 93% accuracy.
  • Users can draw digits directly in the browser using a responsive canvas.
  • The backend is built with Flask and TensorFlow to deliver instant predictions.
  • The web application is built using HTML5, CSS3, and JavaScript and designed for responsiveness and modern aesthetics.
  • Ideal for educational use, machine learning portfolios, and real-world digit classification tasks.
